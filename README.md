# @startupteam/components

> React Components

[![NPM](https://img.shields.io/npm/v/@startupteam/components.svg)](https://www.npmjs.com/package/@startupteam/components) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save @startupteam/components
```

## Usage

```jsx
import React, { Component } from 'react'

import { TestDrawer } from '@startupteam/components'

class Example extends Component {
  render() {
    return <TestDrawer />
  }
}
```
